## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Python 3 (for local server)

### Installation & Running Locally

```bash
# Navigate to project directory
cd AgriNex

# Start local server
python3 -m http.server 8888

# Open browser to http://localhost:8888
```
